# 🌦️ Weather Data Analysis

A Python-based weather data analysis project that performs data cleaning, statistical analysis, and visualization of weather datasets, along with a simple GUI built using Tkinter.

---

## 🚀 Features

- Data cleaning and preprocessing  
- Statistical analysis (averages, trends, etc.)  
- Data visualization:
  - Bar plots  
  - Scatter/Dot plots  
  - Heatmaps  
- User-friendly GUI using Tkinter  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Tkinter  

---

## 📊 What This Project Does

This project takes raw weather data and:
- Cleans and processes the dataset  
- Computes useful statistics like averages  
- Generates visual insights using graphs and plots  
- Provides a simple GUI interface for interaction
